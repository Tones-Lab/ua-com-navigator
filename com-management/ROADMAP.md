# Roadmap

This roadmap tracks high‑level workstreams. Items are grouped by status rather than week‑based phases.

## Completed

- Multi‑server sessions (basic + certificate auth)
- FCOM browsing, favorites, and search
- Friendly/Raw preview with inline editing
- Overrides management and metadata
- Builder (literal/eval/processor) and undo/redo
- Overview dashboard with vendor statistics
- MIB browser endpoints and UI shell

## In progress

- Documentation consolidation
- UI polish and accessibility improvements

## Planned

- Cross‑server diff and promotion workflow
- Expanded test automation
- CI/CD deployment guidance
- Schema update automation
- PCOM COM Navigator UI (dedicated PCOM workflows)
- PCOM creation UI/helper (guided UI + scripted creation)
- Legacy rule to COM conversion helper (scripted migration)

## Future

- Bulk promotion and batch editing
- Advanced analytics for overrides and processor usage
- Additional PCOM authoring workflows
