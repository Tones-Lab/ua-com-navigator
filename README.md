# UA COM Navigator

Central workspace for the COM Management application and supporting assets.

## What’s in this repository

- com-management/ — Full-stack COM Management app (backend + frontend)
- coms/ — COM rule content used for indexing, search, and local tooling
- architecture/ — Architecture plans, API specs, and design notes
- scripts/ — Helper scripts for maintenance and tooling
- ssl/ — Local SSL assets (if used)
